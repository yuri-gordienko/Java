1. Порахувати кількість кутів в N трикутниках і K ромбах.
N і K передаються параметрами функції, де 0 < N, K < 10.000.
Функція повинна повернуту сумарну кількість кутів.
Приклад:
Вхід: 1, 1
Вихід: 7

2. Функція приймає три цілих цисла, при цьому два з них однакові.
Треба повернути те, яке відмінне від інших або ж повернути -1, якщо всі три числа однакові.
Приклад:
Вхід: 1, 1, 3
Вихід: 3

3. На вхід приймаємо три числа. Треба віднімати від першого числа друге до тих пір, поки отриманий пезультат
не стане меншим за третє число і після цього вже повернути результат.
Приклад:
Вхід: 20, 3, 10
Вихід: 8

4. На вхід функції приходить масив цілих чисел, а другим параметром - його розмір.
Знайдіть в масиві перше число, яке оточене нулями. Якщо таке число не знайдено, поверніть -1,
інакше поверніть індекс такого елемента.
Приклад:
Вхід: [0, 4, 0, 8], 4
Вихід: 1
